OpenrowDB **0.1.8** redesigns the workspace UI — connection info moves to the title bar, the action bar collapses search to an icon, and pagination adapts to sidebar width.

### Changed

- **Connection header → title bar** — connection name and `driver · user@host:port/db` subtitle now live in the macOS window title bar. The inline 44pt header bar is removed, giving more vertical space to data. Transfer menu and read-only badge moved to the window toolbar. Disconnect moved to sidebar right-click context menu.
- **Action bar redesign** — search collapses to a magnifying-glass icon by default; click to expand with spring animation (ESC or blur-when-empty collapses). Column filter becomes a single icon button with a popover; turns accent-colored when a filter is active.
- **Adaptive pagination** — uses `ViewThatFits` to switch between full layout (Page N of M · range · picker) and compact layout when the sidebar is wide.
- **Icon button sizing** — all icon-only buttons use explicit `16×16` frame so `.glass` style applies uniform padding everywhere.
- **Sidebar expand state** — `ConnectionNode` owns its own `@State var expanded` instead of a parent `Set<UUID>` binding, removing the last cross-wire risk between same-named connections.
- **Transfer menu** — renders as a standard macOS toolbar popup button.

### Fixed

- **Pagination bar hidden under glass sidebar** — added `leadingInset` offset so nav buttons and page-info text are not obscured by the translucent NavigationSplitView overlay.
