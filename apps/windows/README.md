# apps/windows

Windows port — **not started yet.** Targeted post v1.

Likely candidates for the UI framework:
- WinUI 3 (native Windows feel)
- Avalonia (cross-platform .NET, less Microsoft-y)
- Tauri (reuse web tech — would conflict with our "no Electron-ish" rule)

Decision will be made after v1 ships and we have feedback from real Mac users.

In the meantime: if you want to help scope this, open a discussion at https://github.com/ryan-prayoga/openrowdb/discussions
